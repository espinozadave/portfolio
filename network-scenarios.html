Network Security Incident Reports

Overview

This page presents scenario-based network security activities completed during the Connect and Protect: Networks and Network Security course. Each activity demonstrates applied knowledge in analyzing network attacks, implementing security best practices, and responding to incidents in a controlled, simulated environment.

Scope

  • Network hardening – identifying vulnerabilities and recommending security measures
  • Network attack analysis – reviewing network layer attacks and identifying indicators of compromise
  • OS hardening techniques – understanding system vulnerabilities and recommending mitigation strategies
  • Network communication analysis – evaluating TCP/IP layer communications and detecting potential risks
  • Incident response using NIST CSF – applying the NIST Cybersecurity Framework to respond to a simulated security incident

Key Learning & Skills Demonstrated

  • Critical analysis – identifying risks, attack patterns, and system weaknesses
  • Incident response planning – evaluating appropriate measures to prevent and respond to network attacks
  • Risk identification & mitigation – recommending hardening and defensive strategies based on scenario analysis
  • Applied scenario-based problem solving – making decisions based on given logs, network traffic patterns, and breach scenarios
  • Communication of findings – documenting incident reports clearly and professionally

Network Security Scenarios

Analyze Network Layer Communication

Reviewed DNS and ICMP traffic using network data and identified protocols to detect potential malicious activity in transit.
Analyze Network Attacks

Investigated a website service interruption scenario, identified likely causes, and analyzed the impact of network attacks on organizational operations.
Apply OS Hardening Techniques

Examined a simulated website security incident, reviewed tcpdump logs, and recommended OS-level security measures to prevent similar attacks.
Analysis of Network Hardening

Assessed a fictional social media company’s vulnerabilities and proposed network hardening strategies to prevent future breaches.
